# Function-Visualizer

## Scop
Vizualizator interactiv pentru functii f(x,y) si extensii 3D.

## Structura
- src: cod sursa
- data: date de intrare
- figures: exporturi
- docs: documentatie

## Pornire
1. Deschide MATLAB.
2. Seteaza folderul curent la proiect sau ruleaza `startup`.
3. Ruleaza `runVisualizer` din Command Window.

## Recomandari
- Foloseste operatori elementwise `.^ ./ .*` in expresii.
- Pastreaza Nmax rezonabil pentru performanta.
